# python-projects-Student Record Management System Using File Handling
 

Abstract

The Student Record Management System is a Python-based application developed to manage student records efficiently using file handling techniques. The system allows users to add, view, search, and delete student details. Student information such as roll number, name, and marks is stored in a text file, providing permanent storage without the need for a database. This project demonstrates the use of Python functions, loops, conditional statements, and file operations.

Objectives

- To store student records using file handling.
- To provide easy access to student information.
- To perform add, search, view, and delete operations efficiently.

Features

- Add Student Record
- View All Student Records
- Search Student by Roll Number
- Delete Student Record
- File-Based Data Storage

Technologies Used

- Python
- File Handling
- Functions
- Loops
- Conditional Statements

Conclusion

This project provides a simple and effective solution for managing student records using Python file handling. It helps beginners understand real-world applications of file operations and data management.
